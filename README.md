# tfk-api-forms
hapi plugin for Tfk API

## Installation

From npm

```
$ npm install tfk-api-forms
```

## Setup
Fill in information of your system in /config.
Be sure to have your mongoDB running.
Run setup from your terminal.

```
$ npm run setup
```

## Server
